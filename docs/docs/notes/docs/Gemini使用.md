如果您身处国内，那么无论是注册Gemini的API Key还是在萌译中使用该功能，可能**需要全程开启VPN**，且VPN的节点不能处于香港和澳门。  
若需查看更多Gemini API支持的地区，请[点击此处](https://ai.google.dev/gemini-api/docs/available-regions)。

:::
